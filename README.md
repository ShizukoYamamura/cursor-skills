# My Cursor Skills

Cursor Agent で使う個人用スキル集。`~/.cursor/skills/` に配置して使用する。

## スキル一覧

| コマンド | スキル名 | 概要 |
|---|---|---|
| `/code-review` | code-review | C#/Unity コードの品質レビュー |
| `/deep-research` | deep-research | あらゆるトピックの深掘りリサーチ → Markdownレポート出力 |
| `/html-article` | html-article | リサーチ＋セマンティックHTML5記事の執筆 |
| `/web-design` | web-design | モダンなWeb UI（HTML/CSS）の構築 |
| `/web-review` | web-review | HTML/CSSのコード品質を辛口レビュー |

## 使い方

Cursor の Composer（Ctrl+I）またはチャットで `/コマンド名` を入力して呼び出す。

## インストール

```bash
git clone https://github.com/<your-username>/cursor-skills.git ~/.cursor/skills
```
